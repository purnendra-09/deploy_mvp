About Smart Attendance

Smart Attendance is a modern, QR-based attendance management system designed for teachers. It simplifies student tracking, class organization, and attendance analytics — all within a clean, mobile-friendly interface.

Features

  • QR-based student identification and scanning
  • Real-time attendance tracking with Firestore
  • Class-level and student-level analytics
  • Print-ready QR codes for physical distribution
  • Multi-teacher data isolation (each teacher sees only their data)
  • Mobile-responsive design with hamburger navigation
